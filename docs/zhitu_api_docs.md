# Zhitu API 完整文档

> 更新时间: 2026-02-18（基于官网 https://www.zhituapi.com/hsstockapi.html 重新整理）
> Base URL: `https://api.zhituapi.com`
> 股票代码格式: 纯代码（如 `000001`）或带市场后缀（如 `000001.SZ`），具体见各接口说明

---

## 请求频率总限制

| 版本 | 每分钟上限 | 备注 |
|------|-----------|------|
| 体验版 | 1,000次 | 通用接口 |
| 包量版 | 300次 | 通用接口 |
| 包月版 | 1,000次 | 通用接口 |
| **包年版** | **3,000次** | 通用接口；部分接口另有独立限制见下文 |
| 至尊版 | 6,000次 | 通用接口 |
| 企业版 | 不限 | 仅企业版专属接口（独立子域名） |

> ⚠️ `/hs/public/realall` 和 `/hs/custom/realall` 在包年版/至尊版下还有**每分钟最多1次**的额外限制（独立于通用速率池）

## 一、基础列表类

### 1.1 股票列表
- **URL**: `/hs/list/all`
- **描述**: 获取基础的股票代码和名称
- **更新频率**: 每日16:20

| 字段 | 类型 | 说明 |
|------|------|------|
| dm | string | 股票代码，如：000001 |
| mc | string | 股票名称，如：平安银行 |
| jys | string | 交易所，sh=上证，sz=深证 |

### 1.2 概念指数列表（券商数据）
- **URL**: `/hs/list/sectors`
- **描述**: 获取概念指数代码和名称
- **更新频率**: 每日16:20

| 字段 | 类型 | 说明 |
|------|------|------|
| dm | string | 概念指数代码，如：101076.BKZS |
| mc | string | 概念指数名称，如：GN玻璃 |
| jys | string | 交易所 |

### 1.3 一级市场板块列表（券商数据）
- **URL**: `/hs/list/primary`
- **更新频率**: 每日16:20

| 字段 | 类型 | 说明 |
|------|------|------|
| mc | string | 一级市场名称，如：1000SW1基础化工 |

### 1.4 板块明细列表（券商数据）
- **URL**: `/hs/sectors/{板块指数名称}`
- **描述**: 依据《一级市场板块列表》获取对应板块的股票明细
- **更新频率**: 每日16:20

## 二、股池监控类 ⭐核心功能

> 所有股池：日期格式 `yyyy-MM-dd`，历史数据从 **2019-11-28** 开始，更新频率：**交易时间段每10分钟**

| zj | number | 封板资金（元） |
| zbc | number | 炸板次数 |

## 三、实时行情类

> 实时行情分两类数据源：**公开数据（每1分钟更新）** 和 **券商数据（真实时）**
>
> | 接口类型 | URL | 更新频率 | 集成状态 | 备注 |
> |---------|-----|---------|---------|----|
> | 公开-单只 | `/hs/real/ssjy/{code}` | 1分钟 | ✅ | 无额外限制 |
> | 公开-全市场 | `/hs/public/realall` | 1分钟 | ✅ | 包年版每分钟**最多1次** |
> | 公开-多只 | `/hs/public/ssjymore` | 1分钟 | ✅ | 最多20只 |
> | 券商-单只 | `/hs/real/time/{code}` | **实时** |  | 比公开数据更新更快 |
> | 券商-全市场 | `/hs/custom/realall` | **实时** |  | 包年版每分钟最多1次 |
> | 券商-多只 | `/hs/custom/ssjymore` | **实时** |  | 最多20只 |

### 3.1 实时交易（公开数据-单只） 
- **URL**: `/hs/real/ssjy/{股票代码}`
- **描述**: 单只股票实时交易数据（可理解为日线最新数据）
- **更新频率**: 交易时间每1分钟

| 字段 | 类型 | 说明 |
|------|------|------|
| p | number | 当前价格（元） |
| pc | number | 涨跌幅（%） |
| ud | number | 涨跌额（元） |
| o | number | 开盘价（元） |
| h | number | 最高价（元） |
| l | number | 最低价（元） |
| yc | number | 昨收价（元） |
| v | number | 成交量（手） |
| cje | number | 成交额（元） |
| zf | number | 振幅（%） |
| hs | number | 换手率（%） |
| lb | number | 量比 |
| pe | number | 市盈率（动态） |
| sjl | number | 市净率 |
| sz | number | 总市值（元） |
| lt | number | 流通市值（元） |
| zs | number | 涨速（%） |
| fm | number | 5分钟涨跌幅（%） |
| zdf60 | number | 60日涨跌幅（%） |
| zdfnc | number | 年初至今涨跌幅（%） |
| t | string | 更新时间 |

### 3.2 实时交易（公开数据-全市场） 
- **URL**: `/hs/public/realall`
- **描述**: 一次性获取所有股票实时数据（同 3.1 字段 + `dm` 股票代码）
- **更新频率**: 交易时间每1分钟
- **限制**: 至尊版/包年版可用，最多 **每分钟1次**
- **系统处理**: `realtime_sync.py` 用60秒内存缓存应对此限制

### 3.3 实时交易（公开数据-多只） 
- **URL**: `/hs/public/ssjymore?stock_codes=代码1,代码2,...`
- **描述**: 指定最多20只股票的实时数据
- **更新频率**: 交易时间每1分钟
- **限制**: 至尊版/包年版可用，最多 **20只**

### 3.4 实时交易（券商数据-单只） 
- **URL**: `/hs/real/time/{股票代码}`
- **描述**: 单只股票实时交易数据（券商通道，比公开数据更新更快）
- **更新频率**: **实时**
- **返回字段**: 同 3.1，包含 `lb`量比、`hs`换手率等实时指标

### 3.5 实时交易（券商数据-全市场） 
- **URL**: `/hs/custom/realall`
- **描述**: 一次性获取所有股票实时数据（券商通道）
- **更新频率**: **实时**
- **限制**: 至尊版/包年版可用，最多 **每分钟1次**

### 3.6 实时交易（券商数据-多只） 
- **URL**: `/hs/custom/ssjymore?stock_codes=代码1,代码2,...`
- **描述**: 指定最多20只股票券商实时数据
- **更新频率**: **实时**
- **限制**: 至尊版/包年版可用，最多 **20只**

### 3.7 买卖五档盘口（新增） 
- **URL**: `/hs/real/five/{股票代码}`
- **描述**: 实时买卖五档盘口数据
- **更新频率**: **实时**

| 字段 | 类型 | 说明 |
|------|------|------|
| ps | number | 委卖价 |
| pb | number | 委买价 |
| vs | number | 委卖量 |
| vb | number | 委买量 |

### 3.8 当天逐笔交易
- **URL**: `/hs/real/zbjy/{股票代码}`
- **描述**: 当天逐笔交易数据，按时间倒序
- **更新频率**: 21:00

| 字段 | 类型 | 说明 |
|------|------|------|
| d | string | 日期（yyyy-MM-dd） |
| t | string | 时间（HH:mm:ss） |
| v | number | 成交量（股） |
| p | number | 成交价 |
| ts | number | 方向（0:中性，1:买入，2:卖出） |

## 四、历史数据类

### 4.1 历史分时交易 
- **URL**: `/hs/history/{代码.市场}/{级别}/{除权}?st=开始&et=结束`
- **举例**: `/hs/history/000001.SZ/d/n?st=20250101&et=20250430`
- **级别**: `5` `15` `30` `60` `d`(日) `w`(周) `m`(月) `y`(年)
- **除权**: `n`(不复权) `f`(前复权) `b`(后复权) `fr`(等比前) `br`(等比后)
- **日期格式**: YYYYMMDD 或 YYYYMMDDhhmmss
- **更新频率**: 分钟级盘中实时更新；日线及以上每日 **15:30 开始更新**，预计 17:10 完成

| 字段 | 类型 | 说明 |
|------|------|------|
| t | string | 交易时间 |
| o | float | 开盘价 |
| h | float | 最高价 |
| l | float | 最低价 |
| c | float | 收盘价 |
| v | float | 成交量 |
| a | float | 成交额 |
| pc | float | 前收盘价 |
| sf | int | 停牌（1停牌，0不停牌） |

### 4.2 最新分时交易 
- **URL**: `/hs/latest/{代码.市场}/{级别}/{除权}?limit=条数`
- **举例**: `/hs/latest/000001.SZ/d/n?limit=20`
- **描述**: 获取最新 N 条 K 线数据，交易时间升序
- **更新频率**: 实时

### 4.3 历史涨跌停价格
- **URL**: `/hs/stopprice/history/{代码.SZ或.SH}?st=开始&et=结束`
- **描述**: 获取历史涨跌停价格
- **更新频率**: 每日 0点

| 字段 | 类型 | 说明 |
|------|------|------|
| t | string | 交易日期 |
| h | float | 涨停价格 |
| l | float | 跌停价格 |

### 4.4 行情指标
- **URL**: `/hs/indicators/{代码.SZ或.SH}?st=开始&et=结束`
- **描述**: 获取量比、涨速、多日涨幅、换手等行情指标，可和局为历史数据
- **更新频率**: 每日 **16:30 开始更新**，预计 20:00 完成

| 字段 | 类型 | 说明 |
|------|------|------|
| lb | float | 量比 |
| om | float | 1分钟涨速(%) |
| fm | float | 5分钟涨速(%) |
| 3d | float | 3日涨幅(%) |
| 5d | float | 5日涨幅(%) |
| 10d | float | 10日涨幅(%) |
| 3t | float | 3日换手(%) |
| 5t | float | 5日换手(%) |
| 10t | float | 10日换手(%) |

### 4.5 企业版历史数据【支持1m级别】❌仅企业版
- **URL**: `http://{专属子域名}.zhituapi.com/hs/hsstock/vip/{代码.SZ或.SH}/{级别}/{除权}?st=&et=&lt=`
- **举例**: `http://xxx.zhituapi.com/hs/hsstock/vip/000001.SZ/1/n?st=20250101&et=20250201`
- **描述**: 企业专属服务器直接对接券商数据源，支持 **1分钟** K线；分钟级历史数据最近**3年**内，日线及以上为上市以来全量
- **级别**: `1` `5` `15` `30` `60` `d` `w` `m` `y`
- **更新频率**: 实时更新
- **请求频率**: 不限（企业专属服务器）
